# PIZZA

## An interactive webpage for building a pizza order.

## By _Sailor Winkelman_

## Description

_This webpage uses checkboxes to collect a user's pizza preferences, and determine
the price of the pizza._

### Specifications

_User Selects pizza size._

* Example Input: "small"
* Example Output: "size = small"

_User selects pizza toppings:_

* Example Input: "pepperoni," "pineapple," and "onion" are checked
* Example Output: toppings = "pepperoni", "pineapple", "onion"

_Pizza object is created for the order when the form is submitted._
* Example Input: "submit"
* Example Output: var pizza1 = new Pizza { size, toppings, price  
}

_Price is calculated based on pizza toppings and pizza size._
* Example Input: User checks "large," "pepperoni," and "olive."
* Example Output: Size: Large
                Toppings: pepperoni, olive  
                Price: $13.50


## Setup/Installation Requirements

* _Open a web browser_
* _Type in: https://sailor27.github.io/pizza/_
* _Type your name in where it says "Enter your name"_
* _Select your desired pizza toppings_
* _Click "submit order"_

## Known Bugs
* If you select two sizes of pizza, the size will default to large.
* The function to add $1 for every pizza topping works only inside front-end logic.
